# Fighter Aircraft Status System
## Usage
### 1. Install
```bash
$ git clone 'This repository'
$ cd fighter-aircraft-status-system
$ cd fighter-aircraft-backend
$ npm install or yarn
$ cd ../
$ cd fighter-aircraft-frontend
$ npm install or yarn
```

### 2. Run
```bash
$ cd fighter-aircraft-backend
$ npm start or yarn start
```
```bash
$ cd fighter-aircraft-frontend
$ npm dev or yarn dev
```
